## Ember CLI implementation for Customer.io

### Usage

`ember install:addon ember-cli-customerio`


### Configuration

In your environment configuration ensure that you set the site-id and
api-key for the CIO object:

```
    CIO: {
      'site-id': '',
      'api-key': ''
    }
```
